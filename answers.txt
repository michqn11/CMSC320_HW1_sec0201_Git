Q1: Why might we consider using Git?

Answer: We might consider using git so that we can maintain numerous versions of the same project and so that we can easily
rollback changes or even create a custom mix of previous additions and versions.

and combat things like infectious diseases.


Q3: What is the difference between Merging and Rebasing?

Answer: Merging preserves the history of both branches that are combined and creates a merge commit.
Rebasing adds the commits of one branch on top of the commits of the main branch, creating a single, linear
history of commits.
